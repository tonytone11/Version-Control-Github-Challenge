1. Explain what the terminal is in your own words.

	The terminal is a built in application that allows the user to create, access, and manipulate files and folders on a computer. User will be able to do tasks by entering built-in commands.

2. What is the major difference between Git and GitHub?

	Git is a tool created to help developers see/track changes made to their code. Meanwhile GitHub, is a platform that uses git providing a space for more developers to share and edit projects online.

3. What are the steps to fork a GitHub repository?

	1st step: Locate and click the fork button
	2nd step: Create and name your new fork
	3rd step: Double-check that the fork created is attached to your account
